webpack中vue的路径问题

# 基于webpack搭建的vue引用资源的路径问题

# template资源路径问题

  1. 使用纯路径

    用:src = "require(''@/assets/'')"

    或者import bg from '@/assets/...'然后在data里 url:bg

    或者用相对路径

  2. 用纯变量

    <img :src="require('@/' + url)" alt="" />
    1

    前面至少要'@/' 或者更长

  3. 用字符串加变量

    <img :src="require('@/assets/avatars/' + item + '.png')" alt="" />
    1

    require(path) ,path 至少要有三部分组成, 目录+文件名+后缀

    目录 => webpack 才知道从哪里开始查找 后缀 => 文件后缀，必须要加上，不然会报错 文件名 => 可用变量表示

# css图片路径

background: url(~@/assets/cover_images/外婆.jpg);
1
  • base64图片直接放src就可以
