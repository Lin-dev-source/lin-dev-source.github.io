# 关于toggle组件

在监听点击的时候，如果页面在加载，会触发el-loading，表面会覆盖一层".el-loading-mask, .el-loading-spinner",所以要把他们忽略掉，

当我把加载设置为点击后立即加载时，出现的el-loading-parent--relative使我的toggle立即消失，将里面加载的东西设置为requestWithoutInterceptors即可
